논문 제목 : "PCL : Proxy-based Contrastive Learning for Domain Generalization"

# 문제점
1. Domain Generalization의 도전 과제
  - 여러 도메인에서 학습한 모델이 보지 못한 목표 도메인에서도 일반화되도록 하는 것은 어려운 문제
  - 일반적으로 사용하는 대조 학습 방법론은 domain-invariant (도메인이 바뀌더라도 변하지 않는) 표현을 학습하는데, 이는 주로 샘플 간의 의미적 관계를 활용
2. 기존 대조 학습의 한계
  - 대조 학습은 positive pair 간의 거리를 좁히고, negative pair 간의 거리를 넓히는 방식으로 작동
  - 이러한 접근 방식은 도메인 일반화 문제에 바로 적용될 때 효과적이지 않다는 것이 발견
  - 도메인이 다를 경우, 동일한 클래스의 샘플 간에도 큰 차이가 있을 수 있으며, 이는 대조 학습의 성능을 저하시킬 수 있음
  
# 해결책
  1. Proxy 기반 대조 학습 방법 제안
   - 본 논문에서는 기존의 샘플 간 관계를 proxy와 샘플 간 관계로 대체하는 새로운 대조 학습 방법을 제안한다.
   - Proxy는 각 클래스의 대표 벡터로, 도메인 간 차이를 줄이는 데 도움이 된다.

  2. Proxy 사용의 장점
   - Proxy를 사용함으로써, 각 클래스의 특징을 대표하는 벡터를 학습할 수 있다.
   - Proxy와 샘플 간의 관계를 학습하여 도메인 불변 (domain-invariant) 특성을 효과적으로 학습할 수 있다.
   - 이 방법은 도메인이 바뀌더라도 변하지 않는 특징을 더 잘 추출할 수 있게 한다.

  3. 구체적인 접근 방식
   - 모델이 각 클래스에 대해 Proxy 벡터를 학습
   - Positive pair는 동일한 클래스의 샘플과 Proxy, negative pair는 다른 클래스의 샘플과 Proxy로 설정하여 학습을 진행
   - 이를 통해 샘플 간의 직접적인 관계 대신, 각 샘플이 속한 클래스의 Proxy와의 관계를 통해 학습이 이루어짐

* proxy 벡터란?
     - 각 클래스의 특징을 대표하는 벡터로, 대조 학습에서 사용되는 중요한 개념\
