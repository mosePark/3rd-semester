"Context-aware Attentional Pooling for Fine-grained Visual Classification"

# 요약
  - 문제점 : 미세한 시각적 분류에서 동일 하위 범주 내 상당한 변화와 다른 하위 범주 간 미묘한 차이를 효과적으로 포착하는 문제
  - 해결책 : 논문에서 제시하는 CAP 방법으로 해결
    → 문제점의 의미? : 사람이 주관적으로 판단하기 매우 어려운 이미지 분류 문제(이름은 다르지만 굉장히 비슷하게 생긴 새)를 제안하는 방법론으로 캡쳐

# 접근 방식
  - CAP은 기본 CNN의 출력을 입력으로 사용하고, 다양한 크기와 위치에 있는 여러 영역을 통해 컨텍스트 정보를 효율적으로 학습
  - Bilinear pooling을 사용하여 다양한 크기의 영역을 고정 크기의 특성 벡터로 변환
  - A context-aware attention mechanism이 관련성 높은 영역에 집중하도록 하여 전체적인 컨텍스트 정보를 생성
  - 공간 구조 인코딩은 영역들 사이의 공간적 배열을 캡처하기 위해 recursive network를 사용

# 실험
  - 8개의 벤치마크 FGVC 데이터셋
  - 미세한 시각적 차이를 구별하는 데 매우 효과적
  - 모델의 복잡성과 inference 시간 평가
  - t-SNE

느낀점 : 미세한 차이를 컴퓨터 비전을 통해 찾아내는 과정이 굉장히 신기했고 attention 메커니즘이 성과가 어느 분야에서도 매우 훌륭하다는 것을 다시 느꼈습니다.
