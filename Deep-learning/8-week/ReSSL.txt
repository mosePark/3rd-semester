논문 제목 : "ReSSL: Relational Self-Supervised Learning with Weak Augmentation"

# 문제점과 해결책
  - 문제점 : self-supervised learning의 대부분 연구는 같은 인스턴스에서만 고민, 다른 인스턴스 간의 관계 모델링의 필요
  - 해결책 : ReSSL 제안, 다른 인스턴스 간의 relation을 모델링

# 소개
  ## 기존 방법
    - Contrastive Learning : 서로 다른 뷰 간의 유사성을 학습하는 방법으로, Noise Contrastive Estimation을 사용하여 구현
    - Instance Discrimination : 널리 채택된 사전 텍스트 작업으로, 동일한 이미지의 다른 뷰 간의 관계를 학습

  ## 새로운 접근
    - 인스턴스 간의 관계를 활용 : 데이터의 다양한 증강 사이에서 쌍별 유사성의 일관성을 유지하는 것을 목표
    - 약한 증강 : 공격적인 데이터 증강보다 신뢰할 수 있는 관계를 형성하는 것이 중요
    → 목표 분포의 강화: 모델이 중요한 데이터 포인트 간의 관계를 더욱 잘 이해하고 덜 중요한 관계는 덜 강조하도록 도움
    → 메모리 버퍼와 모멘텀 업데이트 네트워크 활용: 학습 효율성과 성능을 향상시키는 데 기여

# 방법론
  1. 이미지로부터 2가지 증강된 view 생성하고 embedding 진행
  2. 증강된 두 임베딩과 배치 내 다른 임베딩들고의 유사도를 계산해 관계 분포를 생성
  3. 두 관계 분포의 LK-divergence를 최소화하여 두 뷰 간의 관계적 일관성을 유지하도록 함
  * 모멘텀 업데이트 네트워크 : 과거 배치에서의 특징 임베딩을 저장하기 위한 큰 메모리 버퍼를 유지
  * 타겟 분포의 강화 : tau를 사용해 타겟분포를 뾰족하게 만듬 = z와 가장 유사한 feature를 강조

느낀점 : 기존에 연구되는 방향과 전혀 다른 아이디어로 접근해 논문을 작성한 것이 굉장히 독창적이고 대단하다고 느꼈음. 그리고 인스턴스 간의 관계를 분포 관점에서 loss를 풀어낸 것이 재밌었음.
